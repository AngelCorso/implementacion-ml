# Implementación de una técnica de aprendizaje máquina sin el uso de un framework.

En esta implementación se utiliza el conjunto de datos caloriesPrediction.csv, cojunto obtenido durante el 2do semestre con el fin de predecir el número de calorías de un alimento mediante el análisis de su información nutrimental con las variables:
- Y: Cantidad de calorias (kcal)	
- X1: Cantidad de grasas (gr)	
- X2: Cantidad de proteinas (gr)	
- X3: Cantidad de carbohidratos (gr)

Al ejecutar el programa se puede observar el RMSE (Root Mean Square Error) del modelo y una comparación de 10 observaciones de los datos esperados y los datos predichos.

Se decidió utilizar RMSE para contemplar mejor el error en cuanto a calorías se refiere.
